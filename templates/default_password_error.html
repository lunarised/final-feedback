Security Warning

The admin panel is using the default password.

For security reasons, you must change the admin password in your .env file before the application will be fully functional.

To fix this:
  • Open your .env file in the application directory
  • Find or add the line:
    ADMIN_PASSWORD=admin123
  • Replace admin123 with a strong, unique password:
    ADMIN_PASSWORD=your-secure-password-here
  • Save the file and restart the application

This is a development/configuration issue, not a bug. Using a strong, unique password protects your feedback data from unauthorized access.
